# 🍰 AneDoces - Sistema de Gestão de Pedidos e Orçamentos

Sistema completo desenvolvido para gerenciamento de clientes, pedidos e orçamentos para uma confeitaria, com integração via WhatsApp e aplicação mobile.

## 🚀 Tecnologias utilizadas

* ASP.NET Core Web API
* .NET MAUI (App Mobile/Desktop)
* Entity Framework Core
* SQL Server
* C#

## 📱 Funcionalidades

### 🔹 Clientes

* Cadastro de clientes
* Listagem de clientes
* Integração com WhatsApp direto pelo app

### 🔹 Pedidos

* Listagem de pedidos
* Visualização de detalhes

### 🔹 Orçamentos

* Criação de orçamentos
* Aprovação de orçamento
* Conversão automática em pedido
* Envio de orçamento via WhatsApp

### 🔹 Busca Geral

* Pesquisa por nome do cliente
* Retorna clientes, pedidos e orçamentos juntos

### 🔹 Integração WhatsApp

* Geração automática de mensagem
* Link direto para envio via WhatsApp

## 🧠 Arquitetura

* Backend em API REST
* Frontend em .NET MAUI
* Comunicação via HTTP (JSON)
* Organização em camadas (Models, Services, Pages)

## 🎯 Objetivo do projeto

Aplicar conceitos reais de desenvolvimento full stack com C#, integrando backend e aplicação mobile, simulando um sistema real de vendas e atendimento.

## 🚧 Próximas melhorias

* Cadastro de pedidos pelo app
* Dashboard com indicadores
* Melhorias visuais (UI/UX)
* Autenticação de usuários
